# 🚀 Exercícios de Lógica de Programação (Visualg)

Este repositório contém uma coleção de algoritmos básicos desenvolvidos em **Visualg** (.alg) para praticar fundamentos de programação, como variáveis, operadores matemáticos e estruturas condicionais (SE/ENTÃO/SENÃO).

## 📂 Projetos no Repositório

Aqui está uma síntese de cada desafio resolvido:

1.  **Soma:** Entrada de 3 valores e exibição do resultado da adição.
2.  **Volume:** Cálculo matemático para determinar o volume com base em medidas fornecidas.
3.  **Comissão:** Sistema de cálculo de vendas onde a taxa muda conforme o valor do produto (5% para produtos acima de R$ 5000 e 10% para valores menores).
4.  **Idade para dirigir:** Verificador simples que libera ou bloqueia a permissão com base na maioridade legal.
5.  **Senha:** Algoritmo de validação de acesso para verificar se a senha digitada está correta.
6.  **Notas:** Sistema escolar que classifica o aluno em **Aprovado** (>= 7), **Recuperação** (entre 4 e 7) ou **Reprovado** (<= 4).
7.  **Voto:** Verificador de obrigatoriedade de voto por idade:
    * **Acima de 70 anos:** Não vota.
    * **Entre 18 e 70 anos:** Vota.
    * **Menores de 18 anos:** Não vota.

## 💻 Tecnologias
* **Visualg 3.0** (Linguagem Portugol)
*  **Flowgorithm** (Para a criação dos fluxogramas)
